# Installation

## Download

```shell
$ git clone https://github.com/ShioCMS/shio-gogeta.git
$ cd shio-gogeta
```

## Deploy 

### 1. Compile

Use Maven to compile Shio CMS Gogeta.

```shell
$ mvn clean install
```
